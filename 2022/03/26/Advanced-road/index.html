测试开发工程师的进阶之路

发表于 2022-03-26 更新于 2022-10-22 分类于 求职

古语有言，居安思危，思则有备，有备无患。于是本篇给出了测试开发工程师的进阶指南，愿诸君都能得偿所愿，不负坚持。

一. 江浙沪相关企业社招要求概述

(A) 大疆科技

工作职责

  1. 参与测试自动化平台(含自动化测试框架、工具软件、智能测试系统等)的需求分析、方案设计、功能开发和部署，优化相关业务的特性和性能；
  2. 参与测试自动化平台建设，包括通用框架、公共技术组件、接口抽象和公共技术工具；
  3. 参与制定测试自动化平台相关技术演进的 Roadmap，以及实施计划；
  4. 负责嵌入式软件平台测试的自动化测试开发设计和执行；
  5. 负责嵌入式软件平台持续集成与系统优化。

任职要求

  1. EE/CS 或相关专业的本科以上学历，2年及以上工作经验；

  2. 熟悉 Python/C/C++/Java/JavaScript/LabVIEW 等编程语言中的一种或多种；

  3. 良好的学习能力及团队合作能力，责任心强，喜欢挑战，对技术有追求；

  4. 较好的系统软件开发和集成经验，熟悉操作系统、计算机组成原理；

  5. 具备以下一种或多种技能或经验者优先：

    (a) 丰富的 Linux 应用程序开发经验，熟练运用 Linux 的同步互斥机制；

    (b) 参与过 DevOps 整体解决方案实现，有 Iaas/Paas/Saas 平台开发经验；

    (c) 理解 RTOS 的运行机制，能够独立开发设计 RTOS 的应用；

    (d) 有 Linux 内核开发经验，如: Linux 内核的移植和开发，Linux 设备驱动程序的设计和开发，良好的问题定位能力；

    (e) 理解硬件和 ARMv7、ARMv8 架构；

    (f) 精通 MFC 或 QT GUI 软件开发；

    (g) 熟练使用 Flask/Django/Tornado/SpringBoot 等一种或多种Web开发框架，熟悉 MySQL、Mongodb 等数据库及 RabbitMQ/Kafka 等消息列队组件；

    (h) 熟悉 Vue/Electron/React/Angular 等前端框架的一种或多种；

  6. 良好的学习能力及团队合作能力，责任心强，喜欢挑战，对技术有追求。

(B) TP-LINK

系统测试工程师

岗位职责

  1. 包括但不限于无线及有线路由器，交换机，CloudSDN，智能家庭设备，IPCamera，终端接入设备，云平台，APP 的测试工作。
  2. 主要负责制定测试方案并定义测试标准，编写测试用例及自动化测试脚本，开发测试工具。
  3. 参与新测试方法及新测试工具的引入，跟踪市场及工厂反馈并推动产品的设计优化，跟踪及推动产品的安全性设计优化，为企业及运营商客户提供测试支持服务。
  4. 不断优化质量体系并最大程度地保证项目质量，提升客户满意度。

任职要求

  1. 大学本科/硕士学历(优先)，一年及以上测试相关经验。计算机，网络工程，电子，通信，信息安全相关理工类专业。
  2. 对技术有热情并有较好的技术服务意识，喜欢研究新的技术热点，能刻苦钻研并热衷动手解决问题。
  3. 有责任心，严谨性及较强的逻辑思维能力，良好的表达沟通能力及基本的英语口语表达能力。
  4. 熟练掌握至少一种计算机语言，精通 Python，C/C++ 相关开发技能者优先。
  5. 有测试开发经验，测试机构相关从业经验，海外运营商测试经验及英语口语表达能力优秀者优先。

(C) 哔哩哔哩

资深测试开发工程师

职位描述

工作职责：

  1. 对测试工具，测试平台和测试方法进行优化，包括但不限于开发新的测试工具，平台，并不断优化以提高工作效率；
  2. 结合业务现状和行业前景，思考质量提升，效率优化方案，指引团队测试技术的发展方向；
  3. 提升整体团队技术氛围，落地培养整体团队代码审查以及编写能力；并在半年内培养白盒测试团队；
  4. 大型项目期间，主导并保证前端和后端的性能指标符合期望，并产出完整清晰的性能报告以及给出合理的测试建议；
  5. 总结提炼软件测试工作经验，指导中/高级测试开发工程师的技术工作；
  6. 定期学习新的测试技能和工具，与团队成员共同分享，定期进行技术培训。

任职要求

  1. 大学本科或以上学历(优先)，需要计算机及相关专业；
  2. 五年以上测试开发工作经验，以及三年以上软件开发、运维等相关工作经验，熟悉软件测试基本理论、流程和方法；
  3. 有很强的分析设计能力和代码能力，熟悉 Java、C++、Python 等至少一种编程语言；
  4. 有足够健全的质量保障思路和经历，擅长自动化，有移动端和稳定性方面的优先；
  5. 熟悉 Linux/Unix 系统；
  6. 熟悉常用数据库 MySQL、Oracle 等的基本操作，能熟练编写 SQL；
  7. 具备独立完成系统设计、搭建、推进实施的能力，有 Web 应用开发经验者为佳。

加分项

  1. 具备良好的沟通能力，工作主动积极，有团队合作精神；
  2. 在测试领域具备很强多项专业技能者优先，例如性能、自动化、安全等；
  3. 熟悉电商行业，有电商相关行业从业经验者优先。

(D) 字节跳动

测试开发工程师

职位描述

  1. 参与 IT 部门企业级应用产品的测试开发。通过有效的工具、有效的指标衡量并提高整体工程质量，提升产品力；
  2. 与开发团队一起协作，针对客户端/服务器产品进行测试/性能分析；
  3. 参与产品 CI/CD，利用测试保障全部环境的自动化与可靠性；
  4. 设计开发测试/性能分析工具，构建部门的测试/性能分析体系。

职位要求

  1. 有大型互联网架构下的服务端测试、开发经验，对于常见分布式架构有了解；
  2. 熟悉 Javascript/Java/C++/Python 等常用语言，有良好的算法功底；
  3. 了解常见测试组件，如 Jest/JUnit/GTest；
  4. 了解 Docker/K8S，以及 Travis/Jenkins 等 CI/CD 工具。

(E) 米哈游

资深测试开发工程师

职位描述

岗位描述

  1. 负责业务系统的测试工作，改进现有测试中的不足；
  2. 负责自动化用例建设，提升项目测试自动化程度；
  3. 负责测试相关工具或平台开发，提升测试效率及平台化程度；
  4. 负责业务系统的专项(性能、安全、协议等)测试工作；
  5. 负责研究新型测试技术，帮助项目提升整体质量。

职位要求

  1. 计算机相关专业本科以上学历(优先)，1 年以上测试开发、QA、开发测试；
  2. 熟悉以下任意领域的技术：游戏引擎、游戏&工具开发、数据库、Web 前端、Web 后台；
  3. 熟练某个编程语言如：Python、Golang、Lua、C++、C#、Javascript 等；
  4. 了解某个测试领域：安全、客户端性能、服务器性能、自动化等；
  5. 具备系统化开发能力，有创新及探索精神，善于发现项目不足并实行有效的解决方案；
  6. 掌握系统测试的方法与原理，能够针对功能模块的编写并执行测试用例；
  7. 思路清晰，善于交流，能与设计开发团队进行有效的沟通；
  8. 了解各类型游戏，有较强的游戏操作能力，对游戏有独到的见解和敏锐的观察力；
  9. 具备游戏行业经验优先。

(F) 哲库科技

ATE 测试开发工程师

职位描述

  1. 与研发及应用工程师一起为新产品设计芯片 ATE 测试方案，并按照计划进行测试项目开发和量产导入；包括前期的可测性评估和测试方案制定，探针卡及测试接口板的设计验证，测试程序的开发与维护，量产导入与量产问题解决。
  2. 优化测试程序提高测试覆盖率，提升量产良率，减少测试时间。

任职要求

  1. 电子信息工程、微电子、电气自动化及仪器仪表等相关专业本科及以上学历(优先)；
  2. 熟悉数字电路、模拟电路、信号系统原理；
  3. 熟悉手机、电视、电源管理类芯片测试，高速、RF 芯片测试背景更佳；有 ATE 测试平台 Ultraflex，93K 开发经验者优先，3 年以上工作经验；
  4. 熟悉 C、C++、VB、Perl 等编程语言，具有较强的现场调试经验；
  5. 了解硬件 PCB 设计理论知识和制造流程；
  6. 具备良好的沟通能力和团队协作精神；英语水平良好；
  7. 较强的逻辑思维和解决问题的能力，对流程善于思考并敢于正确表达看法；
  8. 有良好的产品质量意识，具有较强责任心。协助公司新产品量产导入开发及验证，符合产品市场需求。

二. 技能向进阶指南

  1. 深入学习编程语言，包括 Python，C++，Java 等
  2. 学习常见测试组件，如 Jest/JUnit/GTes 等；精进某个测试领域，如安全、客户端性能、服务器性能、自动化
  3. 学习 Linux/Unix 系统
  4. 学习 MySQL、Oracle 等数据库，熟练编写 SQL
