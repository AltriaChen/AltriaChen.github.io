从本篇开始，阿伟同学将集中精力整理自己的 Python 学习笔记。Python 学习笔记 (零一) 篇，将主要介绍 Python 中的六个基本数据类型，以及对应的常用方法。除此之外，还介绍了 Python 中的选择语句与循环语句。

一. Python中的标准数据类型

Python3 中有六个标准的数据类型：
Number (数字)，String (字符串)，List (列表)，Tuple (元组)，Set (集合)，Dictionary (字典)
这六个标准数据类型中，
不可变数据 (3个)：Number (数字)，String (字符串)，Tuple (元组)；
可变数据 (3个)：List (列表)，Dictionary (字典)，Set (集合)。

(A) Number(数字)

(a) 简介

Python3 支持 int，float，bool，complex.

Python 中运算符：
+ 加；- 减；* 乘； / 除以(结果可以是浮点数)；// 除以(结果是整型)； % 求余数，例如 30%4 结果为 2；** 幂运算，例如 3**2 结果为 9.

运算符优先级从高到低为：
幂运算 正负号 算术操作符 比较操作符 逻辑操作符。
